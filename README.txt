Zet deze complete map in GitHub of upload de inhoud naar Netlify.

Belangrijk:
1. In Supabase Authentication > URL Configuration:
   - Site URL = jouw Netlify domein
   - Redirect URL = jouwdomein/update-password.html
2. Structuur verwacht:
   - css/theme.css
   - js/supabase.js
   - js/app.js
   - js/auth.js
3. Scanpagina ondersteunt:
   - batch inslag
   - batch uitslag
   - QR scanner
   - nieuwe producten aanmaken vanuit scan
   - SKU basis = alles vóór laatste -0000
